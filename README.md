# kolbo-catalog

קטלוג מוצרים של Kolbo Online

## ניהול האתר

לעריכת תוכן האתר, כולל כותרות, תיאורים, מחירים ותמונות:
1. פתח את `admin.html` בדפדפן
2. הכנס את ה-GitHub Personal Access Token שלך
3. בדוק חיבור ל-GitHub
4. ערוך מוצר קיים או הוסף מוצר חדש
5. עדכן את השדות: שם מוצר, תיאור קצר, תיאור מלא, מחיר, תיקיית תמונות, תכונות וגלריית תמונות
6. לחץ "עדכן ודחף ל-GitHub"

### הוספת מוצר חדש
- לחץ "הוסף מוצר חדש"
- מלא שם, תיאור קצר, תיאור מלא, מחיר, תיקיית תמונות והעלה תמונות
- הוסף תכונות באופן שרשומה אחת = תכונה אחת
- לחץ "שמור מוצר" ואז "שמור ופרסם" - זה ייצור עמוד מוצר חדש ויעדכן את הקטלוג

### יצירת GitHub Token

1. עבור ל-GitHub Settings > Developer settings > Personal access tokens
2. צור token חדש עם הרשאות `repo`
3. העתק את ה-token והכנס בדף הניהול

## מבנה האתר

- `index.html` - עמוד ראשי עם קטלוג מוצרים
- `product-*.html` - עמודי מוצר בודדים
- `admin.html` - דף ניהול
- `scripts/` - סקריפטים לעדכונים
- תיקיות עם תמונות מוצרים
